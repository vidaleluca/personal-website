# Static assets

Drop your files here:

- `profile.jpg` — your profile photo (used in the About section). Optimized: 984×1200, 125 KB, progressive JPEG. A `profile.webp` (58 KB) is also shipped for automatic format negotiation.
- `og.png` — Open Graph preview image, 1200×630. Shown when the site is shared on social platforms.

Both files are referenced in the code. Without them the About image will show a fallback block and the OG preview will 404 on share.
